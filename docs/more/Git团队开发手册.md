# Git团队开发手册
## 提交到本地仓库 - Commit
```shell
git add .
git commit -m'NEW:新增xxx功能'
```
### 提交备注规范：
> NEW：新增了xxx功能
> MOD：优化了xxx功能/逻辑
> BUG：修复了xxx的bug

## 推送到远程仓库（gitee）- push
```shell
# git push origin [branch]
git push origin czq
```
## 拉取远程仓库代码 - pull
```shell
# git pull origin [branch]
git pull origin czq
```
